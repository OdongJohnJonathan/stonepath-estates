Stonepath Estates

Stonepath Estates is a web-based real estate listing platform designed to simplify property buying, selling, and renting in Uganda. This platform aims to provide a transparent, reliable, and user-friendly space for property owners, buyers, and renters to connect directlyâ€”eliminating the need for costly or dishonest brokers.

Features

- Property Listings: Browse detailed listings with photos, descriptions, and pricing.
- Advanced Search & Filters: Find properties by location, price range, type (rent/sale), and other criteria.
- User Registration & Profiles: Secure sign-up and personalized profiles for buyers, sellers, and agents.
- Admin Dashboard: Manage listings, users, and inquiries efficiently.
- Order Tracking & Communication: Buyers can track property inquiries and communicate with sellers.
- Mobile Money Payments: Integrated support for popular Ugandan payment methods (MTN, Airtel Money).
- Responsive Design: Fully usable on desktops, tablets, and smartphones.

Technology Stack

- Frontend: HTML5, CSS3, JavaScript
- Backend:PHP
- Database:MySQL (via XAMPP)
- Version Control: Git & GitHub

Installation & Setup

1. Clone the repository: git clone https://github.com/OdongJohnJonathan/stonepath-estates.git
2. Import the database schema from `stonepath_estates.sql` using phpMyAdmin or MySQL CLI.
3. Configure the database connection in `admin/includes/connect.php`.
4. Run the project on your local server (e.g., XAMPP Apache server).
5. Access the platform via `http://localhost/stonepath-estates` in your browser.

Usage

- Register as a user to list properties or make inquiries.
- Use the search and filters to find your ideal property.
- Admins can log in to manage and moderate listings.
